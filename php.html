Panduan Pemrograman PHP untuk Pemula hingga Mahir

Pendahuluan

PHP (Hypertext Preprocessor) adalah bahasa pemrograman yang sangat populer di dunia web. PHP digunakan untuk membuat situs web dinamis dan aplikasi web. Salah satu alasan mengapa PHP sangat populer adalah karena PHP dapat dijalankan pada berbagai sistem operasi dan platform, serta memiliki kemampuan untuk berinteraksi dengan berbagai jenis database, seperti MySQL.

Dalam artikel ini, kita akan membahas PHP mulai dari pengenalan dasar, instalasi, penulisan kode sederhana, hingga konsep-konsep lanjutan yang akan membantu Anda membangun aplikasi web yang dinamis menggunakan PHP.

Apa Itu PHP?

PHP adalah bahasa pemrograman server-side yang digunakan untuk membuat halaman web dinamis. Berbeda dengan HTML dan CSS yang bersifat client-side (dijalankan di browser), PHP dijalankan di server dan mengirimkan hasilnya ke browser pengguna.

PHP dapat berfungsi untuk melakukan berbagai tugas, mulai dari membuat formulir, mengelola data pengguna, hingga menghubungkan aplikasi web dengan database.

PHP pertama kali diciptakan oleh Rasmus Lerdorf pada tahun 1993 dan secara resmi dirilis pada tahun 1995. PHP terus berkembang seiring waktu dan saat ini menjadi salah satu bahasa pemrograman web paling banyak digunakan di dunia. 

        <?php
            echo "Halo, Dunia!";
        ?>

Contoh kode di atas akan mencetak "Halo, Dunia!" di halaman web ketika dijalankan.

Instalasi PHP

Untuk mulai menulis kode PHP, Anda perlu menginstal server web dan PHP pada komputer Anda. Ada beberapa cara untuk melakukan ini, salah satunya adalah dengan menggunakan paket pengembangan seperti XAMPP, WAMP, atau MAMP, yang menyediakan server Apache, PHP, dan MySQL dalam satu paket.

  • XAMPP: Tersedia untuk Windows, Linux, dan macOS.
  • WAMP: Hanya tersedia untuk Windows.
  • MAMP: Tersedia untuk macOS dan Windows.

Setelah Anda menginstal salah satu dari paket tersebut, Anda dapat mulai menulis dan menguji kode PHP di localhost.

Struktur Dasar PHP

Untuk menulis kode PHP, Anda harus menggunakan tag PHP khusus yang dimulai dengan <?php dan diakhiri dengan ?>. Semua kode PHP harus ditulis di dalam tag ini agar server mengenalinya sebagai kode PHP. 

        <?php
            echo "Ini adalah kode PHP pertama saya!";
        ?>

Anda bisa menulis kode PHP di dalam file dengan ekstensi .php dan kemudian mengakses file tersebut di server web Anda (misalnya http://localhost/nama-file.php).

Variabel dan Tipe Data

Dalam PHP, variabel digunakan untuk menyimpan data. Variabel PHP dimulai dengan tanda dolar ($), diikuti dengan nama variabel. Nama variabel harus diawali dengan huruf atau garis bawah dan hanya boleh berisi huruf, angka, dan garis bawah.

Mendeklarasikan Variabel

        <?php
            $nama = "John Doe";
            $umur = 25;
            echo $nama;
            echo $umur;
        ?>

Dalam contoh di atas, variabel $nama menyimpan sebuah string, dan variabel $umur menyimpan sebuah angka.

Tipe Data di PHP

PHP mendukung beberapa tipe data dasar, termasuk:

  • String: Teks, misalnya: "Halo Dunia"
  • Integer: Angka bulat, misalnya: 25
  • Float: Angka desimal, misalnya: 3.14
  • Boolean: Nilai benar (true) atau salah (false)
  • Array: Sekumpulan nilai yang terurut
  • Object: Objek yang dibuat berdasarkan kelas
  • NULL: Tidak ada nilai

Fungsi dalam PHP

Fungsi dalam PHP digunakan untuk mengelompokkan kode yang dapat digunakan kembali. Fungsi bisa menerima parameter dan mengembalikan nilai.

Mendeklarasikan Fungsi

        <?php
            function greet($nama) {
                echo "Halo, $nama!";
            }
            greet("Alice");
        ?>

Dalam contoh di atas, fungsi greet menerima parameter $nama dan mencetak pesan selamat datang.

Fungsi yang Mengembalikan Nilai

        <?php
            function tambah($a, $b) {
                return $a + $b;
            }
            $hasil = tambah(5, 3);
            echo $hasil;
        ?>

Pengkondisian (Conditionals)

Pengkondisian digunakan untuk membuat keputusan dalam program. Di PHP, kita dapat menggunakan if, else, dan elseif untuk menentukan jalur eksekusi kode berdasarkan kondisi. 

        <?php
            $angka = 10;
            if ($angka > 5) {
                echo "Angka lebih besar dari 5";
            } elseif ($angka == 5) {
                echo "Angka sama dengan 5";
            } else {
                echo "Angka lebih kecil dari 5";
            }
        ?>

Perulangan (Loops)

Perulangan digunakan untuk mengeksekusi blok kode beberapa kali. PHP mendukung beberapa jenis perulangan seperti for, while, dan foreach.

Perulangan dengan for

        <?php
            for ($i = 0; $i < 5; $i++) {
                echo $i;
            }
        ?>

Perulangan dengan while

        <?php
            $i = 0;
            while ($i < 5) {
                echo $i;
                $i++;
            }
        ?>

Menghubungkan dengan Database MySQL

Salah satu fitur kuat PHP adalah kemampuannya untuk berinteraksi dengan database, seperti MySQL. Anda bisa menggunakan PHP untuk mengambil, menyimpan, dan memanipulasi data dari database. 

        <?php
            $conn = new mysqli("localhost", "username", "password", "nama_database");
            if ($conn->connect_error) {
                die("Koneksi gagal: " . $conn->connect_error);
            }

            $sql = "SELECT id, nama FROM users";
            $result = $conn->query($sql);

            if ($result->num_rows > 0) {
                while($row = $result->fetch_assoc()) {
                    echo "id: " . $row["id"] . " - Nama: " . $row["nama"] . "
";
                }
            } else {
                echo "Tidak ada hasil";
            }

            $conn->close();
        ?>

Kesimpulan

PHP adalah bahasa pemrograman server-side yang powerful dan sangat berguna dalam pengembangan aplikasi web dinamis. Dengan menguasai dasar-dasar PHP seperti variabel, fungsi, pengkondisian, dan perulangan, Anda bisa mulai membuat aplikasi web yang lebih kompleks.

Setelah menguasai dasar-dasar ini, Anda bisa melanjutkan untuk mempelajari konsep lanjutan seperti pengelolaan sesi, pengamanan aplikasi, dan penggunaan framework PHP seperti Laravel atau Symfony.
